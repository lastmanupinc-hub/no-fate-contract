# Documentation

This folder contains supporting documentation for the No Fate ecosystem.

## Contents

### VERIFICATION.md
Complete instructions for verifying canonical document integrity using SHA-256 hashes.

### GPG_SIGNING.md
Complete instructions for GPG key generation, document signing, and signature verification.

### RELEASE_TEMPLATE.md
Template for GitHub Release notes with placeholders for version numbers, hashes, and signatures.

## Purpose

These documents support the canonical ecosystem by providing:
- Verification procedures
- Signing workflows
- Release processes
- User guidance

## Modification Policy

Documentation files in this folder may be updated to improve clarity or reflect changes in tooling. However, they must not alter the canonical documents or their verification procedures.

All documentation updates should be committed with clear change descriptions.
