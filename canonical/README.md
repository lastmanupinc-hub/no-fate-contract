# Canonical Documents

This folder contains the immutable canonical versions of all No Fate ecosystem documents.

## Contents

All documents in this folder are:
- Versioned semantically (vX.Y.Z)
- Hash-verified for integrity
- Cryptographically signed
- Licensed under CC BY-ND 4.0
- Immutable once published

## File Naming Convention

All canonical documents follow this naming pattern:

Document_Name_vX_Y_Z.md
Document_Name_vX_Y_Z.pdf

## Included Documents

### The No Fate Contract
Defines the allowed and disallowed behavior for deterministic systems interacting with law, policy, or enforcement.

### The Deterministic Map of Law
A complete classification of where U.S. legal questions are deterministic, interpretive, discretionary, or procedural.

## Modification Policy

No modifications are permitted to canonical documents in this folder. Any changes require:
1. Creation of a new semantic version
2. New hash generation
3. New cryptographic signature
4. Formal release process

See CONTRIBUTING.md in the root directory for details.
